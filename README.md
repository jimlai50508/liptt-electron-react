## 目錄與檔案
- /main: electron main process code
- /renderer: front-end source code
- /resources: 應用程式資源
- /.config: 基本環境設定
- /.vscode: VSCode環境設定
- /.webpack: webpack環境設定
- /docs: 相關文件
- /dist: 編譯後的 electron main 與 renderer webpack bundle
- /release: 編譯後的懶人包位置
