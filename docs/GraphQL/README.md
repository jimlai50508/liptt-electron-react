# GraphQL

一種相較於傳統 RESTful API 的新的前後端溝通模式，使用類似於 json 的一種新式語法來獲取資源。

(功能實驗中)

## Scalar Type

- Int: 32 位有號整數
- Float: 單精度浮點數
- String: UTF-8 字串
- Boolean: true/false

## Object Type

```
type Article {
  id: ID
  text: String
}
```

## Operation

- query
- mutation
- subscription
